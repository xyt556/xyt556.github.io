软件简介

1.1 软件概述

GeoSHAP v4.5.3_Pro 是一个专业的地理空间数据分析和SHAP值解释的综合性工具。该软件集成了机器学习、空间统计分析和模型可解释性分析等多种功能，支持回归和分类任务，提供完整的机器学习工作流程。

1.2 主要特性

  • 空间自相关分析：支持 Moran's I 和 LISA（局部空间自相关）分析
  • 特征工程模块：多项式特征、交互特征、空间滞后等
  • 模型集成：Stacking、Voting、Blending 等集成学习方法
  • 空间交叉验证：考虑空间依赖性的交叉验证方法
  • 不确定性量化：Bootstrap置信区间
  • 批量预测与空间插值：支持栅格数据批量预测和空间插值
  • 报告自动生成：HTML/PDF格式报告
  • 项目保存/加载：完整的工作流程保存和恢复
  • 多语言支持：中英文界面切换
  • 暗色主题支持：多种主题可选
  • 数据质量检查：自动数据质量评估
  • 特征重要性对比分析：多模型特征重要性对比
  • 模型解释性增强：SHAP、LIME支持
  • GPU加速支持：XGBoost GPU加速检测
  • 内存优化：大数据处理优化
  • 样本不平衡处理：SMOTE、ADASYN、过采样/欠采样、类别权重等

1.3 版本信息

  • 版本号：4.5.3_Pro
  • 作者：@3S&ML
  • 开发语言：Python
  • GUI框架：Tkinter / ttkbootstrap

系统要求与安装

2.1 系统要求

最低配置

  • 操作系统：Windows 7/10/11, Linux, macOS
  • Python版本：Python 3.7 或更高版本
  • 内存：4GB RAM（推荐 8GB 或更高）
  • 硬盘空间：至少 2GB 可用空间
  • 处理器：双核处理器（推荐四核或更高）

推荐配置

  • 操作系统：Windows 10/11 或 Linux
  • Python版本：Python 3.8+
  • 内存：16GB RAM 或更高
  • 硬盘空间：10GB 或更多（用于存储数据和结果）
  • 处理器：四核或更多核心
  • GPU：NVIDIA GPU（可选，用于XGBoost加速）

2.2 依赖库安装

必需依赖

pip install ttkbootstrap rasterio geopandas pandas numpy scikit-learn shap matplotlib joblib statsmodels xgboost lightgbm catboost scipy seaborn scikit-optimize pysal esda libpysal jinja2 weasyprint lime tqdm imbalanced-learn

可选依赖

  • GPU支持：需要安装 CUDA 和 cuDNN（用于XGBoost GPU加速）
  • PDF报告生成：需要 weasyprint（可能需要额外系统依赖）

安装注意事项

  1. Windows用户：

    • 建议使用 Anaconda 或 Miniconda 环境
    • 某些库（如 rasterio、geopandas）可能需要从 conda-forge 安装：
      conda install -c conda-forge rasterio geopandas

  2. Linux用户：

    • 可能需要安装系统级依赖：
      sudo apt-get install gdal-bin libgdal-dev python3-gdal

  3. macOS用户：

    • 使用 Homebrew 安装 GDAL：
      brew install gdal

  4. imbalanced-learn兼容性：

    • 如果使用 scikit-learn 1.4+ 版本，可能会遇到兼容性问题
    • 建议使用 scikit-learn 1.3.2：
      pip install scikit-learn==1.3.2

python GeoShapv4.5.3_Pro.py

快速开始

3.1 基本工作流程

  1. 准备数据

    • 准备栅格数据文件夹（包含所有特征栅格文件）
    • 准备样本点数据（Shapefile格式，包含目标变量）

  2. 加载数据

    • 在"Input Data"标签页中指定数据路径
    • 选择目标列（要预测的变量）

  3. 配置选项

    • 在"Options"标签页中选择模型和参数
    • 设置预处理选项

  4. 运行分析

    • 点击"Run Analysis"按钮
    • 等待分析完成

  5. 查看结果

    • 在"Model Evaluation"标签页查看模型评估结果
    • 在"SHAP Analysis"标签页查看特征重要性
    • 在"Spatial Analysis"标签页查看空间分析结果
    • 在"Report"标签页生成完整报告

3.2 示例：回归分析

  1. 数据准备

    • 栅格文件夹：包含多个环境变量栅格（如温度、降水、海拔等）
    • 样本点：包含观测值的点数据，目标列为"target"

  2. 设置问题类型

    • 在"Options"标签页中，选择"Problem Type"为"regression"

  3. 选择模型

    • 勾选要使用的模型（如XGBoost、Random Forest等）

  4. 运行分析

    • 点击"Run Analysis"

  5. 查看结果

    • 查看R²、RMSE等回归指标
    • 查看预测散点图和残差图
    • 查看SHAP特征重要性

3.3 示例：分类分析

  1. 数据准备

    • 与回归分析类似，但目标变量应为分类变量（整数类别）

  2. 设置问题类型

    • 选择"Problem Type"为"classification"

  3. 处理样本不平衡（如需要）

    • 勾选"Handle imbalanced data"
    • 选择处理方法（如SMOTE）

  4. 运行分析

    • 点击"Run Analysis"

  5. 查看结果

    • 查看混淆矩阵、分类报告
    • 查看ROC曲线
    • 查看各类别的准确率、精确率、召回率等

功能模块详解

5.1 Input Data（输入数据）标签页

5.1.1 数据源设置

栅格文件夹（Raster Folder）

  • 功能：指定包含所有特征栅格文件的文件夹
  • 要求：
    • 所有栅格文件应在同一文件夹中
    • 栅格文件应具有相同的空间范围和分辨率
    • 支持常见栅格格式（.tif, .tiff, .asc等）
  • 操作：点击"Browse"按钮选择文件夹

样本点（Sample Points）

  • 功能：指定包含观测值的点数据文件
  • 格式要求：Shapefile格式（.shp文件）
  • 必需字段：
    • 几何信息（点坐标）
    • 目标变量列（要预测的变量）
  • 操作：点击"Browse"按钮选择文件

目标列（Target Column）

  • 功能：指定要预测的目标变量列名
  • 默认值：target
  • 自动检测：点击"🔍 Detect"按钮可自动检测可能的目标列

5.1.2 输出设置

输出文件夹（Output Folder）

  • 功能：指定分析结果的保存位置
  • 默认值：./output
  • 包含内容：
    • 训练好的模型文件
    • 评估图表
    • SHAP分析结果
    • 空间分析结果
    • 报告文件

5.1.3 示例数据生成

功能说明：快速生成示例数据用于测试和学习

参数设置：

  • Samples：样本数量（100-5000）
  • Features：特征数量（3-20）
  • Problem Type：问题类型（regression/classification）
  • Add spatial pattern：是否添加空间模式
  • Classes：分类问题的类别数（2-10）
  • Balanced classes：是否平衡类别分布

使用方法：设置参数后点击"🎲 Generate"按钮

5.1.4 数据预览

功能：预览加载的数据统计信息

显示内容：

  • Column：列名
  • Type：数据类型
  • Non-Null：非空值数量
  • Min：最小值
  • Max：最大值
  • Mean：平均值

刷新：点击"🔄 Refresh Preview"按钮更新预览

5.2 Options（选项）标签页

5.2.1 模型选择

功能：选择要训练的机器学习模型

可用模型：

  • XGBoost：梯度提升树（如果已安装）
  • LightGBM：轻量级梯度提升（如果已安装）
  • CatBoost：分类提升（如果已安装）
  • Random Forest：随机森林
  • Gradient Boosting：梯度提升
  • Extra Trees：极端随机树
  • Histogram Gradient Boosting：直方图梯度提升

快捷操作：

  • Select All：全选所有模型
  • Deselect All：取消全选
  • Select Boosting：仅选择提升类模型
  • Model Parameters：打开模型参数设置窗口

5.2.2 预处理选项

特征标准化（Standardize features）

  • 功能：对特征进行标准化处理
  • 方法选择：
    • Standard (Z-score)：Z-score标准化
    • MinMax (0-1)：最小-最大归一化
    • Robust (IQR)：基于四分位数的鲁棒标准化
    • Normalizer (L2)：L2范数归一化
    • MaxAbs：最大绝对值缩放

特征工程（Feature Engineering）

  • 功能：自动创建新特征
  • 可选方法：
    • Interaction：交互特征
    • Statistical：统计特征（均值、标准差等）
    • Ratio：比率特征
    • Log Transform：对数变换
    • Power：幂变换
    • Binning：分箱
    • Clustering：聚类特征
    • Polynomial：多项式特征

相关性过滤（Correlation filter）

  • 功能：移除高度相关的特征
  • 阈值设置：0.5-0.99（默认0.90）
  • 说明：如果两个特征的相关系数超过阈值，将移除其中一个

VIF过滤（VIF filter）

  • 功能：移除多重共线性特征
  • 阈值设置：2.0-50.0（默认10.0）
  • 说明：方差膨胀因子（VIF）超过阈值的特征将被移除

5.2.3 训练参数

问题类型（Problem Type）

  • regression：回归问题（预测连续值）
  • classification：分类问题（预测类别）

测试集比例（Test size）

  • 范围：0.1-0.4
  • 默认：0.2（20%作为测试集）

随机种子（Random state）

  • 范围：0-999
  • 默认：42
  • 作用：确保结果可复现

交叉验证折数（CV Folds）

  • 范围：2-10
  • 默认：5
  • 说明：K折交叉验证的折数

空间交叉验证（Use spatial cross-validation）

  • 功能：考虑空间依赖性的交叉验证
  • 说明：使用空间聚类方法划分训练/测试集，避免空间自相关导致的过拟合

5.2.4 样本不平衡处理

注意：此功能仅适用于分类问题

启用处理（Handle imbalanced data）

  • 功能：自动处理类别不平衡问题

处理方法（Method）：

  • auto：自动选择最佳方法
  • smote：SMOTE过采样
  • adasyn：ADASYN自适应过采样
  • random_oversample：随机过采样
  • random_undersample：随机欠采样
  • borderline_smote：边界SMOTE
  • svm_smote：SVM SMOTE
  • tomek_links：Tomek Links欠采样
  • edited_nn：编辑最近邻
  • smote_tomek：SMOTE + Tomek Links组合
  • smote_enn：SMOTE + ENN组合
  • class_weight：类别权重调整

同时使用类别权重（Also use class weights）

  • 功能：在重采样基础上同时使用类别权重
  • 说明：可以与重采样方法结合使用

5.2.5 超参数优化

启用优化（Enable hyperparameter optimization）

  • 功能：自动搜索最佳超参数

优化方法（Method）：

  • grid：网格搜索（全面但耗时）
  • random：随机搜索（快速）
  • bayesian：贝叶斯优化（如果已安装scikit-optimize）

迭代次数（Iterations）

  • 范围：5-100
  • 默认：20
  • 说明：仅用于random和bayesian方法

5.2.6 集成方法

Stacking Ensemble

  • 功能：堆叠集成学习
  • 说明：使用多个基模型，然后用元学习器组合预测

Voting Ensemble

  • 功能：投票集成学习
  • 说明：多个模型的预测结果进行投票或平均

5.5 Spatial Analysis（空间分析）标签页

5.5.1 空间自相关分析

功能：分析变量的空间自相关性

Moran's I全局自相关

  • 功能：计算全局Moran's I统计量
  • 参数：
    • Weight Type：权重类型（knn/queen/distance）
    • K值：K近邻数量（用于knn权重）
  • 结果：Moran's I值、p值、z值

LISA局部自相关

  • 功能：计算局部空间自相关
  • 结果：每个样本点的局部Moran's I值

可视化：

  • Moran's I散点图
  • LISA聚类图（HH, HL, LH, LL）

5.5.2 空间权重矩阵

权重类型：

  • knn：K近邻权重
  • queen：Queen邻接权重
  • distance：距离权重

参数设置：

  • K值：K近邻数量
  • 距离阈值：距离权重阈值

5.5.3 空间特征

空间滞后（Spatial Lag）

  • 功能：计算空间滞后变量
  • 说明：反映邻近区域的影响

空间特征创建

  • 功能：自动创建空间相关特征
  • 包括：距离特征、方向特征等

5.6 SHAP Spatial Autocorr（SHAP空间自相关）标签页

5.6.1 功能说明

分析SHAP值的空间自相关性，识别空间聚集模式。

5.6.2 操作步骤

  1. 选择模型
  2. 选择特征
  3. 设置权重参数
  4. 点击"Compute"计算
  5. 查看结果表格和图表

5.6.3 结果解释

  • Moran's I：SHAP值的空间自相关程度
  • p值：统计显著性
  • 空间模式：识别SHAP值的空间聚集区域

常见问题

8.1 数据相关问题

Q: 栅格数据格式要求是什么？ A: 支持常见栅格格式（.tif, .tiff, .asc等），所有栅格应具有相同的空间范围和分辨率。

Q: 样本点数据格式要求是什么？ A: 必须是Shapefile格式（.shp文件），包含点几何和目标变量列。

Q: 如何处理缺失值？ A: 软件会自动处理缺失值，可以选择删除或填充。建议在加载数据前检查数据质量。

Q: 数据量很大，运行很慢怎么办？ A:

  • 减少样本数量（在SHAP分析中使用样本百分比）
  • 减少特征数量（使用相关性过滤）
  • 减少模型数量
  • 关闭超参数优化
  • 使用更快的模型（如LightGBM）

8.2 模型训练问题

Q: 模型训练失败怎么办？ A:

  • 检查数据格式是否正确
  • 检查目标列是否存在
  • 查看日志中的错误信息
  • 尝试减少特征数量
  • 检查样本数量是否足够

Q: 如何选择最佳模型？ A:

  • 查看"Model Evaluation"标签页中的指标对比
  • 对于回归问题，关注R²和RMSE
  • 对于分类问题，关注准确率、F1分数等
  • 考虑交叉验证结果

Q: 如何提高模型性能？ A:

  • 启用特征工程
  • 启用超参数优化
  • 使用集成方法
  • 处理样本不平衡（分类问题）
  • 增加训练数据

8.3 SHAP分析问题

Q: SHAP值计算很慢怎么办？ A:

  • 减少样本百分比（如使用50%或更少）
  • 使用TreeExplainer（对树模型更快）
  • 减少背景样本数量

Q: SHAP值计算失败怎么办？ A:

  • 确保模型已训练完成
  • 检查模型类型是否支持SHAP
  • 查看日志中的错误信息
  • 尝试使用KernelExplainer（较慢但更通用）

Q: 如何解释SHAP值？ A:

  • 正SHAP值表示特征对预测的正向贡献
  • 负SHAP值表示负向贡献
  • 绝对值越大，影响越大
  • 查看依赖图了解特征与SHAP值的关系

8.4 空间分析问题

Q: 空间自相关分析失败怎么办？ A:

  • 检查样本点是否有有效的几何信息
  • 确保样本点数量足够（建议>30）
  • 调整权重参数（K值或距离阈值）

Q: 如何解释Moran's I值？ A:

  • 值接近+1：强正空间自相关（相似值聚集）
  • 值接近-1：强负空间自相关（相异值聚集）
  • 值接近0：无空间自相关（随机分布）
  • 查看p值判断统计显著性

8.5 性能优化问题

Q: 如何加快分析速度？ A:

  • 使用GPU加速（XGBoost）
  • 减少数据量
  • 减少模型数量
  • 关闭不必要的功能
  • 使用更快的模型

Q: 内存不足怎么办？ A:

  • 减少样本数量
  • 减少特征数量
  • 分批处理数据
  • 关闭其他程序
  • 增加系统内存

10.1 术语表

  • SHAP值：SHapley Additive exPlanations，用于解释模型预测的特征贡献值
  • Moran's I：全局空间自相关统计量
  • LISA：Local Indicators of Spatial Association，局部空间自相关指标
  • 空间滞后：邻近区域变量的加权平均
  • 交叉验证：将数据分成多折，轮流作为训练集和测试集
  • 空间交叉验证：考虑空间依赖性的交叉验证方法
  • 集成学习：组合多个模型的预测结果
  • Stacking：堆叠集成，使用元学习器组合基模型
  • Voting：投票集成，多个模型投票决定最终预测
  • SMOTE：Synthetic Minority Oversampling Technique，合成少数类过采样技术

10.2 模型说明

XGBoost

  • 类型：梯度提升树
  • 优点：性能优秀，支持GPU加速
  • 适用：回归和分类

LightGBM

  • 类型：轻量级梯度提升
  • 优点：训练速度快，内存占用小
  • 适用：回归和分类

Random Forest

  • 类型：随机森林
  • 优点：稳定，不易过拟合
  • 适用：回归和分类

Gradient Boosting

  • 类型：梯度提升
  • 优点：性能好
  • 适用：回归和分类

10.3 评估指标说明

回归指标

  • R²（决定系数）：衡量模型解释的方差比例，越接近1越好
  • RMSE（均方根误差）：预测误差的度量，越小越好
  • MAE（平均绝对误差）：预测误差的平均值，越小越好

分类指标

  • 准确率（Accuracy）：正确预测的比例
  • 精确率（Precision）：预测为正例中真正为正例的比例
  • 召回率（Recall）：真正例中被正确预测的比例
  • F1分数：精确率和召回率的调和平均
