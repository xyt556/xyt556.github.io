SHAP分析工具 v5.5.5 功能更新

相对于 v5.5.3 的重大更新

一、模型评估指标增强（v5.5.5 新增）

1. ROC 曲线：AUC 95% 置信区间

  • 所有 ROC 曲线图例中增加 AUC 95% 置信区间 显示
  • 使用 Bootstrap 方法（2000 次重采样）计算
  • 覆盖场景：多模型对比、训练集对比、外部验证、单模型评估
  • 图例示例：AUC = 0.850 (95% CI: 0.782-0.918)

2. 校准曲线：Brier 评分 95% 置信区间

  • 校准曲线图例中增加 Brier 评分及 95% 置信区间
  • 使用 Bootstrap 方法（2000 次重采样）计算
  • 覆盖场景：单模型校准、多模型校准对比、训练集/外部验证校准
  • 图例示例：ModelName (Brier=0.1234, 95% CI: 0.0987-0.1481)

3. ROC 曲线显示优化

  • ROC 曲线坐标轴外扩（xlim/ylim 从 [-0.02, 1.02] 起），避免边线与曲线重叠，图表更清晰

二、SHAP 可视化增强（v5.5.5 新增）

4. 条形图 SHAP 值标注

  • 在 SHAP 条形图（Mean |SHAP value|）上直接标注每个特征的具体数值
  • 数值显示在条形末端右侧，保留 4 位小数
  • 便于在论文或报告中直接引用具体 SHAP 值

三、v5.5.4 功能延续

5. 交互值导出

  • 新增 SHAP 交互值导出按钮（Excel/CSV），位于结果保存区域
  • 仅树模型（随机森林、XGBoost、LightGBM 等）可用
  • Excel 导出包含：交互强度矩阵、交互均值矩阵、Top 交互对、元数据
  • CSV 导出包含：Top 交互对（特征A、特征B、平均绝对交互值、平均交互值）

6. 稳定性与兼容性修复

  • 修复部分模型无 AUC 时导致的 KeyError 等问题
  • 训练集模型对比中，AUC 在“任意模型可用”时展示，缺失模型显示为空白
  • 训练集总览与单指标图支持指标并集，缺失值不标注并给出无数据提示

五、适用场景

科研与论文

  • 使用 AUC 95% CI、Brier 95% CI 满足期刊对置信区间报告的要求
  • 条形图 SHAP 值标注便于在文中引用具体数值
  • 交互值导出便于深入分析与写作

医学与临床

  • Brier 评分及其置信区间有助于评估概率校准质量
  • AUC 置信区间用于模型间差异的统计推断

业务与工程

  • 更稳定的界面与错误处理，减少异常中断
  • 图表信息更丰富，便于汇报与归档
