1. 软件简介

1.1 软件概述

Geographical Detector Pro v6.0.1（地理探测器专业版） 是一款功能强大的地理探测器分析工具，基于地理探测器理论，用于分析地理要素的空间分异性及其驱动因子。软件提供了因子探测器、交互探测器、风险探测器和生态探测器四大核心功能模块，并集成了数据预处理、相关性分析、智能离散化优化等高级功能。

1.2 主要特性

  • ✅ 四大探测器模块：因子探测器、交互探测器、风险探测器、生态探测器
  • ✅ 多数据源支持：Excel/CSV表格数据、栅格数据（GeoTIFF等）
  • ✅ 智能离散化：支持分位数、等间距、标准差、Jenks自然断点等多种方法，并提供自动优化功能
  • ✅ 数据预处理：异常值检测与处理、缺失值填充、数据标准化
  • ✅ 相关性分析：相关性矩阵、VIF（方差膨胀因子）分析
  • ✅ Bootstrap置信区间：提供统计置信区间计算
  • ✅ 项目管理系统：支持项目保存、加载和最近项目快速访问
  • ✅ 结果导出：Excel结果导出、HTML报告生成、图表保存
  • ✅ SCI写作助手：自动生成符合学术论文格式的结果描述文本
  • ✅ 主题切换：支持多种界面主题（需ttkbootstrap支持）
  • ✅ 性能优化：Numba加速计算，大幅提升分析速度

1.3 版本信息

  • 版本号：v6.0.1
  • 发布日期：2025-12
  • 开发团队：@3S&ML Laboratory

    • Excel模式：从下拉菜单选择Y变量，勾选X变量
    • 栅格模式：Y变量已自动加载，点击"➕"添加X变量

  4. 运行分析

4.2 使用自己的数据

Excel/CSV数据格式要求：

  • 文件格式：.xlsx、.xls 或 .csv
  • 数据结构：
    • 第一行为列名（变量名）
    • 第一列为样本ID（可选）
    • Y变量：因变量（数值型）
    • X变量：自变量（数值型或分类型）
  • 数据要求：
    • 无缺失值或已处理缺失值
    • 数值型变量应为连续数值
    • 分类型变量应为整数或字符串

栅格数据格式要求：

  • 文件格式：GeoTIFF（.tif、.tiff）或其他rasterio支持的格式
  • 坐标系统：建议所有栅格使用相同的坐标系统（CRS）
  • 空间分辨率：建议所有栅格具有相同的空间分辨率
  • 数据要求：
    • Y变量栅格：单波段栅格
    • X变量栅格：单波段栅格（每个因子一个文件）

6. 功能详解

6.1 因子探测器（Factor Detector）

功能说明： 因子探测器用于测量各因子对因变量空间分异的解释能力，通过q值（q-statistic）量化因子影响力。

q值含义：

  • q值范围：[0, 1]
  • q值越大，因子对因变量的解释能力越强
  • q值 = 1 - SSW/SST
    • SSW：层内方差和
    • SST：总方差

结果解读：

  • q值：因子解释力（0-1）
  • p值：统计显著性（<0.05为显著）
  • Layers：离散化后的层数
  • CI_lower/CI_upper：Bootstrap置信区间（如果启用）

使用方法：

  1. 加载数据并选择变量
  2. 配置离散化参数（见6.5节）
  3. 勾选"Factor Detection"选项
  4. 点击"Run Analysis"

6.2 交互探测器（Interaction Detector）

功能说明： 交互探测器用于检测两个因子交互作用对因变量的影响，判断交互类型。

交互类型：

  1. 非线性增强（Nonlinear Enhancement）

    • q(X1∩X2) > q(X1) + q(X2)
    • 两个因子协同作用，交互效应超过独立效应之和

  2. 双因子增强（Bi-factor Enhancement）

    • q(X1∩X2) > max(q(X1), q(X2))
    • 交互效应大于任一单独因子

  3. 单因子非线性减弱（Single-factor Nonlinear Weakening）

    • min(q(X1), q(X2)) < q(X1∩X2) < max(q(X1), q(X2))
    • 交互效应介于两个因子之间

  4. 独立（Independent）

    • q(X1∩X2) = q(X1) + q(X2)
    • 两个因子独立作用

  5. 单因子减弱（Single-factor Weakening）

    • q(X1∩X2) < min(q(X1), q(X2))
    • 交互效应小于任一单独因子

结果解读：

  • Factor1/Factor2：交互的两个因子
  • q1/q2：各因子的独立q值
  • q_interact：交互q值
  • Type：交互类型
  • Mechanism：作用机制描述

6.3 风险探测器（Risk Detector）

功能说明： 风险探测器用于识别不同因子层（strata）之间因变量均值的显著差异，检测高风险区域。

统计方法：

  • 使用独立样本t检验比较不同层之间的均值差异
  • 显著性水平：默认0.05

结果解读：

  • Factor：因子名称
  • Layer1/Layer2：比较的两个层
  • Mean1/Mean2：各层的均值
  • p_value：t检验p值
  • Significant：是否显著（Yes/No）

应用场景：

  • 识别高风险区域
  • 比较不同因子水平下的因变量差异
  • 为政策制定提供依据

6.4 生态探测器（Ecological Detector）

功能说明： 生态探测器用于比较不同因子对因变量影响的差异，判断因子作用机制是否相同。

统计方法：

  • 使用F统计量检验两个因子的q值是否存在显著差异
  • 原假设：两个因子的作用机制相同

结果解读：

  • factor_1/factor_2：比较的两个因子
  • F_statistic：F统计量
  • p_value：p值
  • Significant：是否显著不同

应用场景：

  • 比较不同驱动因子的作用强度
  • 识别主要驱动因子
  • 因子重要性排序

6.5 离散化配置

功能说明： 地理探测器要求自变量为分类变量，连续变量需要进行离散化处理。软件提供多种离散化方法和智能优化功能。

离散化方法：

  1. 分位数法（Quantile）

    • 将数据按分位数等分为n组
    • 优点：每组样本数相等
    • 适用：数据分布均匀

  2. 等间距法（Equal Interval）

    • 将数据值域等分为n个区间
    • 优点：区间宽度相等，易于解释
    • 适用：数据分布均匀

  3. 标准差法（Standard Deviation）

    • 以均值为中心，按标准差划分
    • 优点：考虑数据分布特征
    • 适用：数据近似正态分布

  4. Jenks自然断点法（Natural Breaks）

    • 优化组内方差最小化
    • 优点：最大化组间差异
    • 适用：数据有明显聚类特征

离散化配置对话框：

打开方式：

  • 点击参数设置区的"🔧 Config"按钮
  • 或通过菜单：Analysis → Discretization Config...

功能：

  1. 批量设置

    • 选择方法和分组数
    • 点击"Apply to All"应用到所有变量
    • 点击"Apply to Selected"应用到选中变量

  2. 独立配置

    • 双击表格行编辑单个变量
    • 为每个变量设置不同的方法和分组数

  3. 智能优化

    • 点击"Optimize All"自动优化所有变量
    • 点击"Optimize Selected"优化选中变量
    • 系统会自动测试多种方法和分组数，选择最优配置

  4. 配置保存/加载

    • 点击"💾 Save Config"保存配置为JSON文件
    • 点击"📂 Load Config"加载已保存的配置

优化指标：

  • q值：因子解释力（越大越好）
  • GVF（Goodness of Variance Fit）：方差拟合优度（越大越好）
  • 综合评分：综合考虑q值和GVF

6.6 数据预处理

功能说明： 提供数据清洗和预处理功能，提高数据质量。

打开方式：

  • 点击工具区的"🔧 Preprocessing"按钮
  • 或通过菜单：Data → Data Preprocessing...

功能模块：

  1. 异常值检测与处理

    • 检测方法：
      • IQR法（四分位距）
      • Z-score法（Z分数）
      • MAD法（中位数绝对偏差）
    • 处理方法：
      • Remove（删除）
      • Winsorize（缩尾处理）
      • Clip（截断）
      • Median（中位数替换）
      • Mean（均值替换）

  2. 缺失值处理

    • 方法：
      • Mean（均值填充）
      • Median（中位数填充）
      • Mode（众数填充）
      • Interpolate（插值）
      • Value（指定值填充）

  3. 数据标准化

    • 方法：
      • MinMax（最小-最大标准化）
      • Z-score（Z分数标准化）
      • Robust（鲁棒标准化）
      • Log（对数变换）

使用步骤：

  1. 选择要处理的变量（左侧列表）
  2. 查看变量统计信息
  3. 选择处理方法并设置参数
  4. 点击"Detect"或"Apply"执行处理
  5. 点击"Apply Changes"应用所有更改

6.7 相关性分析

功能说明： 分析变量间的相关性和多重共线性，辅助变量选择。

打开方式：

  • 点击工具区的"📊 Correlation"按钮
  • 或通过菜单：Data → Correlation Analysis...

功能：

  1. 相关性矩阵

    • 方法：Pearson、Spearman、Kendall
    • 输出：相关性热力图、相关系数矩阵、p值矩阵
    • 解读：
      • |r| > 0.7：强相关
      • 0.3 < |r| < 0.7：中等相关
      • |r| < 0.3：弱相关

  2. VIF分析（方差膨胀因子）

    • VIF值解读：
      • VIF < 5：无多重共线性
      • 5 ≤ VIF < 10：中等多重共线性
      • VIF ≥ 10：严重多重共线性
    • 建议：VIF ≥ 10的变量考虑删除或进行主成分分析

  3. 完整报告

    • 生成包含相关性矩阵、VIF分析和建议的完整报告
    • 可导出为文本文件

6.8 Bootstrap置信区间

功能说明： 通过Bootstrap重采样方法计算q值的置信区间，评估结果稳定性。

使用方法：

  1. 在参数设置中勾选"Bootstrap CI (slower)"
  2. 或通过菜单：Analysis → Bootstrap CI...
  3. 设置重采样次数（默认1000次）
  4. 运行分析

结果解读：

  • CI_lower：置信区间下限
  • CI_upper：置信区间上限
  • 置信水平：默认95%

注意事项：

  • Bootstrap计算耗时较长，建议样本量较大时使用
  • 重采样次数越多，结果越稳定，但计算时间越长

6.9 敏感性分析

功能说明： 分析不同样本量下结果的稳定性，评估结果的稳健性。

使用方法：

  1. 通过菜单：Analysis → Sensitivity Analysis...
  2. 设置样本量比例范围（默认0.5-1.0）
  3. 设置迭代次数（默认10次）
  4. 运行分析

结果解读：

  • 显示不同样本量下的q值变化
  • 评估结果稳定性
  • 识别关键样本量阈值

6.10 SCI写作助手

功能说明： 自动生成符合学术论文格式的结果描述文本，可直接用于SCI论文的Results部分。

使用方法：

  1. 完成分析后，点击"📝 Generate SCI Results"按钮
  2. 或通过菜单：File → 📝 Generate SCI Results...
  3. 设置因变量名称和显著性水平
  4. 点击"🚀 Generate SCI Text"生成文本
  5. 复制或保存生成的文本

生成内容：

  • Factor Detector结果描述
  • Interaction Detector结果描述
  • Risk Detector结果描述
  • Ecological Detector结果描述
  • 关键发现总结

文本特点：

  • 符合学术写作规范
  • 包含统计显著性说明
  • 提供机制解释
  • 可直接用于论文撰写

7. 操作指南

7.1 完整分析流程

步骤1：准备数据

  1. 整理Excel数据或准备栅格文件
  2. 确保数据格式符合要求（见4.2节）
  3. 检查数据质量（缺失值、异常值等）

步骤2：加载数据

  1. 选择数据源类型（Excel/CSV或Raster）
  2. 加载数据文件
  3. 选择Y变量和X变量

步骤3：数据预处理（可选）

  1. 打开数据预处理对话框
  2. 检查数据统计信息
  3. 处理异常值和缺失值
  4. 必要时进行数据标准化

步骤4：配置离散化

  1. 打开离散化配置对话框
  2. 为每个X变量选择离散化方法
  3. 或使用智能优化功能自动配置
  4. 保存配置（可选）

步骤5：设置分析参数

  1. 选择要运行的分析类型
  2. 设置分组数（如果使用快速设置）
  3. 选择是否计算Bootstrap置信区间

步骤6：运行分析

  1. 点击"▶ Run Analysis"按钮
  2. 等待分析完成（查看进度条）
  3. 查看状态日志了解分析进度

步骤7：查看结果

  1. 在右侧标签页查看各类结果
  2. 查看摘要页面了解整体情况
  3. 查看详细结果表格和图表

步骤8：导出结果

  1. 点击"📊 Export"导出Excel结果
  2. 点击"💾 Charts"保存图表
  3. 或通过菜单导出HTML报告

步骤9：生成SCI文本（可选）

  1. 打开SCI写作助手
  2. 设置参数并生成文本
  3. 复制或保存文本用于论文撰写

7.4 栅格数据处理

加载Y变量栅格

  1. 选择"Raster"数据源类型
  2. 点击"📂 Load Y"按钮
  3. 选择Y变量栅格文件
  4. 软件会自动读取栅格信息和坐标系统

加载X变量栅格

  1. 点击"➕"加载单个X变量
  2. 或点击"➕➕"批量加载多个X变量
  3. 软件会自动对齐栅格（如果坐标系统或分辨率不同）

栅格对齐说明

  • 如果X变量栅格与Y变量栅格的空间分辨率或坐标系统不同，软件会自动重投影和对齐
  • 使用rasterio库时，会进行精确的地理重投影
  • 使用scipy时，会进行简单的缩放对齐

栅格数据要求

  • 所有栅格应为单波段
  • 建议使用相同的坐标系统（CRS）
  • 建议使用相同的空间分辨率
  • 支持GeoTIFF格式（.tif、.tiff）

9. 常见问题

9.1 数据相关问题

Q: Excel文件加载失败怎么办？ A:

  • 检查文件格式是否为.xlsx、.xls或.csv
  • 确保文件未被其他程序占用
  • 检查文件编码（CSV文件建议使用UTF-8编码）
  • 查看控制台错误信息

Q: 栅格数据加载后显示尺寸不匹配？ A:

  • 检查栅格的坐标系统是否一致
  • 检查栅格的空间分辨率是否相同
  • 软件会自动对齐，但建议预处理时统一坐标系统和分辨率

Q: 数据中有缺失值怎么办？ A:

  • 使用数据预处理功能处理缺失值
  • 或使用Excel等工具预先处理
  • 建议在分析前处理所有缺失值

9.2 分析相关问题

Q: 分析运行很慢怎么办？ A:

  • 检查是否安装了Numba库（可大幅提升速度）
  • 减少样本量或变量数量
  • 关闭Bootstrap置信区间计算（如果不需要）
  • 检查计算机内存是否充足

Q: q值都是0或很小怎么办？ A:

  • 检查离散化方法是否合适
  • 尝试使用智能优化功能
  • 检查数据质量（是否存在异常值）
  • 考虑变量选择是否合理

Q: 交互探测器结果中交互类型都是"独立"？ A:

  • 这可能是正常现象，说明因子间确实独立
  • 检查离散化配置是否合理
  • 尝试不同的离散化方法
  • 检查数据是否存在交互关系

9.3 功能相关问题

Q: 如何选择合适的离散化方法？ A:

  • 使用智能优化功能自动选择
  • 根据数据分布特征选择：
    • 数据均匀分布：分位数法或等间距法
    • 数据正态分布：标准差法
    • 数据有明显聚类：Jenks自然断点法
  • 比较不同方法的结果，选择q值最大的方法

Q: Bootstrap置信区间计算失败？ A:

  • 检查样本量是否足够（建议>100）
  • 减少重采样次数
  • 检查内存是否充足

Q: SCI文本生成失败？ A:

  • 确保已完成分析并生成结果
  • 检查结果数据是否完整
  • 查看控制台错误信息

9.4 软件使用问题

Q: 软件启动失败？ A:

  • 检查Python版本是否符合要求
  • 检查依赖库是否安装完整
  • 查看控制台错误信息
  • 检查授权文件是否正确

Q: 界面显示异常？ A:

  • 尝试切换主题（如果支持ttkbootstrap）
  • 检查屏幕分辨率设置
  • 重启软件

Q: 项目文件无法打开？ A:

  • 检查文件是否损坏
  • 检查文件版本是否匹配
  • 尝试使用备份文件

10. 附录

10.1 术语表

  • q值（q-statistic）：地理探测器核心指标，衡量因子对因变量空间分异的解释能力，范围0-1
  • 离散化（Discretization）：将连续变量转换为分类变量的过程
  • 层（Stratum/Layer）：离散化后的分类组别
  • GVF（Goodness of Variance Fit）：方差拟合优度，衡量离散化质量
  • Bootstrap：统计重采样方法，用于估计统计量的置信区间
  • VIF（Variance Inflation Factor）：方差膨胀因子，衡量多重共线性
  • CRS（Coordinate Reference System）：坐标参考系统
