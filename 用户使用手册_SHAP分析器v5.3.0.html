📖 软件简介

什么是SHAP分析器？

SHAP分析器 v5.3.0 是一款基于SHAP（SHapley Additive exPlanations）理论的机器学习模型解释工具。它通过图形化界面，帮助用户：

  • 🔓 打开模型"黑盒"：清晰展示每个特征对预测结果的贡献
  • 📊 生成专业报告：一键导出包含图表、统计信息的完整分析报告
  • 🎨 可视化分析：多种精美的可视化图表，直观展示模型行为
  • 🚀 零代码操作：图形化界面，无需编程基础即可使用

核心特性

  • ✅ 30+种算法支持：涵盖树模型、线性模型、SVM、KNN、神经网络等
  • ✅ 自动问题识别：自动识别分类或回归问题
  • ✅ 智能Explainer选择：根据模型类型自动选择最优的SHAP解释器
  • ✅ 多分类支持：完整支持多分类问题的SHAP分析
  • ✅ 批量模型训练：同时训练多个模型并自动对比
  • ✅ 丰富的可视化：6种SHAP图表类型，满足不同分析需求
  • ✅ 便捷的导出功能：支持模型、图表、报告、数据的多格式导出
  • ✅ 统计信息显示：依赖图支持统计信息可视化（v5.3.0新增）
  • ✅ 多格式图表保存：支持PNG/SVG/PDF/EPS/TIFF等多种格式（v5.3.0新增）
  • ✅ 自定义模型参数：支持为每个算法设置自定义参数（v5.3.0新增）

v5.3.0 新功能亮点

  • 🆕 依赖图统计信息显示：在依赖图中显示特征统计、SHAP统计、相关性等信息
  • 🆕 评估图表多格式保存：支持PNG/SVG/PDF/EPS/TIFF格式，满足不同场景需求
  • 🆕 高级保存选项：支持DPI设置、透明背景、自动裁剪等精细控制
  • 🆕 自定义模型参数：为30+种算法提供示例参数，快速设置推荐配置
  • 🆕 SHAP样本选择优化：支持从全部样本中选择，而不仅限于测试集
  • 🆕 界面布局优化：优化窗口比例，图表显示区域更大
  • 🆕 类别切换自动刷新：多分类问题中切换类别时自动更新图表

💻 系统要求

最低配置

  • 操作系统：Windows 10/11（64位）
  • 内存：4GB RAM（建议8GB以上）
  • 存储空间：500MB以上可用空间
  • 显示器：分辨率1280×720以上

推荐配置

  • 操作系统：Windows 11（64位）
  • 内存：16GB RAM
  • 存储空间：2GB以上可用空间
  • 显示器：分辨率1920×1080或更高

可选依赖

以下依赖为可选，安装后可启用对应功能：

  • XGBoost：用于XGBoost算法（pip install xgboost）
  • LightGBM：用于LightGBM算法（pip install lightgbm）
  • CatBoost：用于CatBoost算法（pip install catboost）
  • scikit-optimize：用于贝叶斯参数优化（pip install scikit-optimize）

🚀 快速开始

第一步：启动程序

  1. 双击运行程序（.exe文件）或运行Python脚本
  2. 程序启动后会自动显示主界面
  3. 首次运行需要进行授权验证（如有）

第二步：加载数据

  1. 点击左侧面板的 "📂 选择数据文件 (CSV/Excel)" 按钮
  2. 在文件选择对话框中选择您的数据文件
  3. 支持格式：.csv、.xlsx、.xls
  4. 加载成功后，界面会显示数据基本信息（行数×列数）

第三步：选择目标变量

  1. 在 "🎯 步骤2: 选择目标列" 区域
  2. 从下拉菜单中选择要预测的目标变量（列）
  3. 程序会自动识别问题类型（分类或回归）

第四步：选择特征列

  1. 在 "✅ 步骤3: 选择特征列" 区域
  2. 勾选要用于训练的特征列
  3. 可以使用 "✓ 全选" 或 "✗ 全不选" 快速操作
  4. 目标变量会自动从特征列中排除

第五步：数据预处理（可选）

  1. 设置训练集比例：默认70%，可通过滑块调整（50%-90%）
  2. 特征缩放：根据需要选择标准化、归一化等方法
  3. 特征选择：如果特征过多，可以使用特征选择方法
  4. 点击 "🔧 数据预处理" 按钮执行预处理

第五步（新增）：自定义模型参数（可选）⭐ v5.3.0新功能

  1. 在 "步骤5.5: 自定义模型参数（可选）" 区域
  2. 选择要设置参数的算法
  3. 点击 "📋 加载示例" 按钮，快速加载该算法的示例参数
  4. 根据需要修改JSON格式的参数
  5. 点击 "💾 保存参数" 保存设置
  6. 这些参数会在训练模型时自动应用

提示：

  • 支持30+种主流算法的示例参数
  • 参数格式为JSON，必须符合JSON语法
  • 自定义参数会覆盖默认参数，但不影响参数优化

第六步：训练模型

  1. 在 "🤖 步骤4: 选择算法" 区域勾选要训练的算法
  2. 可以选择单个算法或批量选择多个算法
  3. 点击 "🚀 训练选中模型" 或 "🏆 训练所有模型"
  4. 等待训练完成，查看模型性能对比

第七步：生成SHAP分析

  1. 在 "📊 SHAP分析" 区域选择已训练的模型
  2. 设置SHAP样本比例（v5.3.0改进：从全部样本中选择，而不仅限于测试集）
    • 树模型：建议100%（计算速度快）
    • 线性模型：建议50-100%（计算速度快）
    • 其他模型：建议10-50%（平衡速度和准确性）
  3. 点击 "⚡ 计算SHAP值" 按钮
  4. 等待计算完成（时间取决于样本数量和模型类型）

第八步：查看可视化结果

  1. 在 "📈 SHAP可视化" 区域选择图表类型
  2. 点击对应的按钮生成图表：
    • 汇总图：查看所有特征的重要性
    • 依赖图：查看单个特征的影响（v5.3.0新增统计信息显示）
    • 交互图：查看特征间的交互作用
    • 瀑布图：查看单个样本的预测过程
    • 力图：查看特征对预测的推动力
    • 条形图：查看特征重要性排序

第九步：导出结果

  1. SHAP图表导出：点击图表上方的 "💾 保存图表" 按钮，选择格式
  2. 评估图表导出：点击 "📊 保存图表 ▼" 下拉菜单，选择格式或高级选项（v5.3.0新增）
  3. 或使用 "📄 导出报告" 导出完整分析报告

📚 详细功能说明

1. 数据加载模块

1.1 支持的数据格式

  • CSV文件（.csv）：逗号分隔值文件
  • Excel文件（.xlsx、.xls）：Microsoft Excel文件

1.2 数据要求

  • 数据格式：第一行为列名（表头）
  • 数据类型：支持数值型和分类型数据
  • 缺失值：程序会自动处理缺失值
  • 编码格式：CSV文件建议使用UTF-8编码

1.3 数据预览

加载数据后，右侧面板会显示：

  • 数据表格预览（前20行）
  • 数据基本信息（行数、列数）
  • 列名列表

2. 数据预处理模块

2.1 训练集比例设置

  • 默认值：70%（训练集70%，测试集30%）
  • 可调范围：50%-90%
  • 推荐值：
    • 小数据集（<1000样本）：70-80%
    • 中等数据集（1000-10000样本）：70%
    • 大数据集（>10000样本）：80-90%

2.2 特征缩放

支持以下缩放方法：

方法 说明 适用场景
标准化 (StandardScaler) 均值0，标准差1 数据近似正态分布
归一化 (MinMaxScaler) 缩放到[0,1]区间 数据有明确边界
Robust缩放 使用中位数和四分位数 数据有异常值
不缩放 保持原始数据 树模型通常不需要

注意：

  • 树模型（随机森林、XGBoost等）通常不需要特征缩放
  • 线性模型、SVM、KNN等通常需要特征缩放

2.3 特征选择

支持以下特征选择方法：

方法 说明 适用场景
RFE 递归特征消除 需要指定特征数量
RFECV RFE + 交叉验证 自动确定最优特征数
SelectKBest 基于统计检验 快速筛选重要特征
SelectPercentile 选择前K%特征 按比例选择特征
SelectFromModel 基于模型重要性 使用模型内置重要性
VarianceThreshold 方差阈值 去除低方差特征

2.4 数据质量检查

程序会自动检查以下问题：

  • ✅ 样本数量：样本过少会给出警告
  • ✅ 特征数量：特征过多会给出建议
  • ✅ 高相关性：检测高度相关的特征对
  • ✅ 缺失值：统计缺失值比例
  • ✅ 数据分布：检查数据分布情况

3.2 自定义模型参数 ⭐ v5.3.0新功能

位置：步骤5.5，位于数据预处理之后、模型训练之前

功能特点：

  • ✅ 支持30+种算法：为每种算法提供示例参数
  • ✅ 一键加载示例：点击"📋 加载示例"快速加载推荐参数
  • ✅ JSON格式：使用标准JSON格式，易于编辑
  • ✅ 参数管理：
    • 💾 保存参数：保存当前算法的自定义参数
    • 🔄 加载已保存：加载之前保存的参数
    • 🗑️ 清除参数：清除当前算法的自定义参数

支持的算法类别：

  • 集成学习：XGBoost、RandomForest、LightGBM、CatBoost、GradientBoosting等
  • 线性模型：Ridge、Lasso、ElasticNet及其CV版本、SGD等
  • 树模型：DecisionTree
  • 其他算法：SVM、KNN、贝叶斯分类器等

使用示例（XGBoost）：

{
  "n_estimators": 100,
  "max_depth": 6,
  "learning_rate": 0.1,
  "subsample": 0.8,
  "colsample_bytree": 0.8,
  "random_state": 42
}

注意事项：

  • 参数必须使用有效的JSON格式
  • 数值参数直接写数字
  • 布尔参数写 true/false
  • 字符串参数用引号括起来
  • 自定义参数会覆盖默认参数，但不影响参数优化

3.3 参数优化

支持三种参数优化方法：

  1. 网格搜索 (GridSearchCV)

    • 遍历所有参数组合
    • 最全面但最耗时
    • 适合参数空间较小的情况

  2. 随机搜索 (RandomizedSearchCV)

    • 随机采样参数组合
    • 速度快，效果通常接近网格搜索
    • 适合参数空间较大的情况

  3. 贝叶斯优化 (BayesSearchCV)

    • 智能搜索最优参数
    • 效率最高，效果最好
    • 需要安装scikit-optimize

注意：如果设置了自定义参数，参数优化会在自定义参数的基础上进行搜索。

3.4 交叉验证

  • K折交叉验证：默认5折
  • 分层K折：分类问题自动使用分层采样
  • 评估指标：自动选择（分类：accuracy，回归：R²）

3.5 模型评估指标

分类任务
  • 准确率 (Accuracy)：正确预测的比例
  • 精确率 (Precision)：预测为正例中真正为正例的比例
  • 召回率 (Recall)：真正例中被正确预测的比例
  • F1分数 (F1-Score)：精确率和召回率的调和平均
  • AUC值：ROC曲线下面积
  • 混淆矩阵：展示分类结果的详细情况
回归任务
  • R²分数：决定系数，衡量模型拟合程度
  • MAE：平均绝对误差
  • MSE：均方误差
  • RMSE：均方根误差
  • 学习曲线：展示训练过程

4. SHAP分析模块

4.1 SHAP值计算

Explainer选择

程序会根据模型类型自动选择最优的Explainer：

Explainer 适用模型 速度 准确性
TreeExplainer 树模型 ⚡⚡⚡ 极快 ⭐⭐⭐⭐⭐ 最高
LinearExplainer 线性模型 ⚡⚡⚡ 极快 ⭐⭐⭐⭐⭐ 最高
KernelExplainer 所有模型 ⚡ 较慢 ⭐⭐⭐⭐ 高
样本比例设置 ⭐ v5.3.0改进

v5.2.2版本：仅支持从测试集中选择样本

v5.3.0版本：支持从全部数据（训练集+测试集）中选择样本

优势：

  • ✅ 可以使用更多样本进行SHAP分析，提高分析结果的稳定性
  • ✅ 对于小数据集，可以使用全部数据进行更全面的分析
  • ✅ 用户可以根据需要灵活选择样本范围

推荐设置：

  • 树模型：建议100%（计算速度快）
  • 线性模型：建议50-100%（计算速度快）
  • 其他模型：建议10-50%（平衡速度和准确性）

注意：样本数量越多，SHAP值越准确，但计算时间越长。

4.2 SHAP可视化图表

📊 汇总图 (Summary Plot)

功能：

  • 展示所有特征的重要性
  • 颜色编码显示特征值大小
  • 直观看出哪些特征影响最大

适用场景：

  • 快速了解特征重要性
  • 发现关键特征
  • 特征筛选

操作步骤：

  1. 计算SHAP值后，点击 "📊 汇总图" 按钮
  2. 图表会显示所有特征的SHAP值分布
  3. 特征按重要性从上到下排序
📉 依赖图 (Dependence Plot) ⭐ v5.3.0增强

功能：

  • 展示单个特征对预测的影响
  • 支持拟合线和95%置信区间
  • 发现特征的非线性关系
  • 🆕 统计信息显示：显示特征统计、SHAP统计、相关性等信息

统计信息内容（v5.3.0新增）：

  • ✅ 特征统计：均值、标准差、中位数
  • ✅ SHAP统计：SHAP值的均值
  • ✅ 相关性：特征值与SHAP值的相关系数
  • ✅ 样本数量：参与计算的样本数
  • ✅ 置信区间：95%置信区间范围（如果启用）

适用场景：

  • 深入分析单个特征
  • 发现特征的非线性效应
  • 特征工程优化
  • 为论文和报告提供统计支撑

操作步骤：

  1. 点击 "📉 依赖图" 按钮
  2. 从下拉菜单选择要分析的特征
  3. 可以设置：
    • 点大小（默认50）
    • 拟合线粗细（默认3.0）
    • 🆕 是否显示统计信息（默认启用）
    • 是否显示置信区间
  4. 使用 "◀ 上一个" 和 "下一个 ▶" 浏览不同特征
  5. 点击 "💾 保存当前图" 或 "💾 保存所有图" 导出

统计信息解读：

  • 相关性（Correlation）：接近±1表示强相关，接近0表示弱相关
  • SHAP均值：绝对值越大，该特征对模型输出的影响越大
  • 特征均值±标准差：了解特征的分布范围
🔄 交互依赖图 (Interaction Plot)

功能：

  • 分析两个特征之间的交互作用
  • 颜色编码显示交互特征的影响
  • 支持拟合线和置信区间
  • 发现隐藏的特征组合效应

适用场景：

  • 发现特征间的交互作用
  • 特征组合优化
  • 深入理解模型行为

操作步骤：

  1. 点击 "🔄 交互图" 按钮
  2. 从下拉菜单选择主特征和交互特征
  3. 可以设置：
    • 点大小（默认30）
    • 拟合线粗细（默认2.0）
    • 是否显示趋势线
    • 是否显示置信区间
  4. 图表会显示两个特征的交互效应
📋 瀑布图 (Waterfall Plot)

功能：

  • 展示单个样本的预测过程
  • 从基准值到最终预测的完整路径
  • 适合向非技术人员解释

适用场景：

  • 解释单个预测结果
  • 向业务人员展示
  • 模型可解释性报告

操作步骤：

  1. 点击 "📋 瀑布图" 按钮
  2. 选择要分析的样本索引（默认第0个样本）

❓ 常见问题解答

Q1: 程序启动失败怎么办？

A: 检查以下几点：

  • 确保操作系统为Windows 10/11（64位）
  • 检查是否有足够的系统权限
  • 查看错误提示信息，根据提示解决问题
  • 如果使用Python版本，确保已安装所有依赖包

Q2: 数据加载失败怎么办？

A: 可能的原因和解决方法：

  • 编码问题：CSV文件建议使用UTF-8编码
  • 格式问题：确保第一行为列名，数据格式正确
  • 文件损坏：尝试用Excel打开文件检查
  • 文件过大：如果文件过大，考虑先进行数据采样

Q3: 模型训练很慢怎么办？

A: 优化建议：

  • 减少特征数量（使用特征选择）
  • 减少训练样本数量（调整训练集比例）
  • 选择计算速度快的算法（如树模型）
  • 关闭参数优化（使用默认参数或自定义参数）
  • 减少交叉验证折数

Q4: SHAP值计算很慢怎么办？

A: 优化建议：

  • 树模型：使用TreeExplainer，可以100%样本
  • 线性模型：使用LinearExplainer，可以50-100%样本
  • 其他模型：减少样本比例（10-30%）
  • 使用更少的特征
  • 关闭交互值计算（如果不需要）

Q5: 如何选择最佳的SHAP样本比例？

A: 建议：

  • 树模型（RandomForest、XGBoost等）：100%（v5.3.0支持从全部样本选择）
  • 线性模型：50-100%
  • SVM、KNN等：10-50%
  • 神经网络：10-30%

原则：在计算时间和准确性之间平衡，样本越多越准确但越慢。

v5.3.0改进：现在可以从全部数据中选择样本，而不仅限于测试集，对于小数据集可以使用全部数据进行更全面的分析。

Q6: 多分类问题如何分析？

A: 程序完全支持多分类：

  • 自动识别类别数量
  • 可以在下拉菜单中选择要分析的类别
  • v5.3.0改进：切换类别时，SHAP图表会自动刷新，无需手动操作
  • SHAP值会为每个类别单独计算

Q7: 导出的图表分辨率够高吗？

A: v5.3.0支持多种格式和分辨率：

  • PNG：位图格式，支持150/300/600 DPI，适合演示
  • PDF：矢量格式，无限放大不失真，适合打印
  • SVG：矢量格式，适合网页展示和编辑
  • EPS：矢量格式，适合学术论文
  • TIFF：高质量位图，支持600 DPI，适合出版

建议：

  • 报告使用PDF或SVG
  • 演示使用PNG（300 DPI）
  • 网页使用SVG
  • 论文使用EPS或SVG
  • 出版使用TIFF（600 DPI）

Q8: 如何处理缺失值？

A: 程序会自动处理：

  • 数值型特征：使用均值填充
  • 分类型特征：使用众数填充
  • 缺失值过多：会给出警告建议

建议：在加载数据前，尽量手动处理缺失值。

Q9: 特征选择后如何恢复？

A: 如果应用了特征选择：

  • 可以重新进行数据预处理，不选择特征选择方法
  • 或者重新加载数据，重新开始

注意：特征选择是不可逆的，建议先备份原始数据。

Q10: 可以同时分析多个模型吗？

A: 可以：

  • 可以训练多个模型并对比性能
  • 但SHAP分析一次只能分析一个模型
  • 需要切换模型并重新计算SHAP值

Q11: 预测新数据时特征不匹配怎么办？

A: 确保：

  • 新数据包含训练时的所有特征
  • 特征名称完全一致（区分大小写）
  • 特征顺序可以不同（程序会自动匹配）
  • 如果缺少特征，需要先补充数据

Q12: 如何理解SHAP值？

A: SHAP值解释：

  • 正值：该特征值使预测结果增加
  • 负值：该特征值使预测结果减少
  • 绝对值大小：表示影响程度
  • 所有SHAP值之和：等于预测值与基准值的差

Q13: 自定义模型参数如何使用？⭐ v5.3.0新功能

A: 使用步骤：

  1. 在"步骤5.5: 自定义模型参数"区域选择算法
  2. 点击"📋 加载示例"加载该算法的示例参数
  3. 根据需要修改JSON格式的参数
  4. 点击"💾 保存参数"保存
  5. 训练模型时，自定义参数会自动应用

注意事项：

  • 参数必须使用有效的JSON格式
  • 数值参数直接写数字，布尔参数写true/false
  • 自定义参数会覆盖默认参数，但不影响参数优化
  • 如果启用参数优化，优化会在自定义参数的基础上进行

Q14: 依赖图的统计信息如何解读？⭐ v5.3.0新功能

A: 统计信息说明：

  • 特征均值±标准差：了解特征的分布范围
  • 中位数：特征的中间值，不受异常值影响
  • SHAP均值：该特征对模型输出的平均影响程度
  • 相关性：特征值与SHAP值的相关系数
    • 接近±1：强相关
    • 接近0：弱相关
  • 样本数：参与计算的样本数量
  • 95%置信区间：如果启用置信区间，会显示置信区间范围

Q15: 评估图表的高级保存选项如何使用？⭐ v5.3.0新功能

A: 使用步骤：

  1. 训练模型并生成评估图表
  2. 点击"📊 保存图表 ▼" → "🎨 高级选项..."
  3. 在对话框中设置：
    • 文件格式（PNG/SVG/PDF/EPS/TIFF）
    • DPI分辨率（150/300/600，仅PNG/TIFF有效）
    • 透明背景（PNG/TIFF支持）
    • 自动裁剪边距（推荐）
  4. 点击"💾 保存"

格式选择建议：

  • 论文：SVG或EPS，300 DPI
  • 演示：PNG，300 DPI，透明背景
  • 打印：PDF或TIFF，300-600 DPI

Q16: 程序占用内存很大怎么办？

A: 优化建议：

  • 减少数据样本数量
  • 减少特征数量
  • 减少SHAP样本比例
  • 关闭不需要的可视化
  • 定期清理内存（关闭并重启程序）

D. 性能优化建议

  1. 数据预处理

    • 提前清理数据，减少缺失值
    • 移除无关特征
    • 对大数据集进行采样

  2. 模型选择

    • 优先选择树模型（计算SHAP值快）
    • 避免使用计算慢的算法（如SVM、KNN）

  3. SHAP计算

    • 树模型使用100%样本（v5.3.0支持从全部样本选择）
    • 其他模型使用10-50%样本
    • 关闭不需要的交互值计算

  4. 内存管理

    • 定期清理不需要的数据
    • 关闭不需要的可视化窗口
    • 分批处理大数据集

E. 错误代码说明

如果遇到错误，程序会显示详细的错误信息。常见错误：

  • "请先训练模型"：需要先训练模型才能进行SHAP分析
  • "请选择模型"：需要在模型选择下拉菜单中选择模型
  • "数据加载失败"：检查数据文件格式和编码
  • "特征不匹配"：预测数据需要包含训练时的所有特征
  • "SHAP计算失败"：可能是模型类型不支持或样本数量不足
  • "JSON格式错误"：自定义参数必须使用有效的JSON格式
